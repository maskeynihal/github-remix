# Github Remix

## Features

- Search Repo
- Get Details of repo

## Tech Used

- Remix

## Installation

### Requirements

- Node >=14
- yarn

### Local Installation

- Clone the repo
  ```sh
  git clone git@github.com:maskeynihal/github-remix.git
  ```
- Change directory
  ```sh
  cd github-remix
  ```
- Install packages
  ```sh
  yarn
  ```
- Run Application
  ```sh
  yarn dev
  ```
